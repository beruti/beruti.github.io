Work Completed on WDI

The WDI Course at General Assembly is an intense 12 weeks, taught by Industry professionals. It provides a hands-on learning environment and covered the fundamentals of full-stack web development. I pitched, worked on and presented 4 projects in an agile environment with daily stand ups.

Danny Fox's Wild West

Shoot 'em up with a twist

A front-end animated game for 2 players inspired by a Cornish Artist and my first gaming experiences with Aladdin on the Sega MegaDrive. Created using HTML, CSS, JavaScript and jQuery it shows what can be built after just two weeks on the course with no prior programming experience.

Cabin Collective

Crowd-sourcing collective build projects

A version of Kickstarter for creating cabins. Users have the chance to pitch features and whole projects as well having the chance to vote what will and will not be built by the community. The site is password- protected, built with Ruby on Rails and uses a SQL database.

The Forum

Examining the power of online-communities and long-form content

We paid homage to Reddit for our first group project and built an app with a Node.js API, a NoSQL database and a jQuery/AJAX front-end. Authentication proved most challenging. The issue was eventually resolved using a combination of JSON Web Tokens and Passport Strategies.

Real Talk

Redefining the floor of the political debate

For the final project we were to build a full-stack app using any tech we liked. I made a chat app specific to the Houses-of-Parliament with the aim of encouraging individuals to engage with the political process. This was done with Sockets, a Node.js API, MongoDB database and an Angular front-end.
